# Linked Node
linked node is a often used data structure in leetcode.

```python

class ListNode:
     def __init__(self, val=0, next=None):
         self.val = val
         self.next = next
```

It is widely used for Tree Search (BFS, DFS) challenge and also in some Two pointers challenge.

- (19) Remove n-th node
- (61) Rotate k times
